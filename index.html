Name Path HTTP Verb Purpose Mongoose Method

Create /cats POST Create a new cat, then redirect somewhere Cat.create()
Show /cats/:id GET Show info about one specific cat Cat.findById()
Edit /cats/:id/edit GET Show edit form for one cat Cat.findById()
